FUNCTIONALITY FOR USER

  1. Registration: User must be Register our self for purchase Product.
  2. Authentication: I'm create proper authentication system for user. Anybody cannot see user information or profile
  3. Login: User can login after Registration from our email id.
  4. Buy Product: There is a button of add cart in every product, If user want any product then they can add the product in their shopping cart.
  5. Shoping Cart: User can add their product in shopping cart and then they can see the Total Amount of the Product if the Amount is out of budget then user can get back or delete own product from cart
  6. Update Profile: User can Update our Profile eg: Name, Number etc.
  7. Change Password: User can Change our Password.
  8. Product Request: If there Isn't any product in website and user want that then user can request to the admin about that
  9. Form Validation: I'm create Proper Validation form from user Registration as well product request

FUNCTIONALITY FOR ADMIN

  1. Seprate Dashboard: Admin have Seprate Dashboard Any user cannot enter in admin section
  2. Add Product: Admin can Add Products.
  3. Edit Product: Admin can Edit Product.
  4. Delete Product: Admin can Delete Product.
  5. User List: Admin can see all Registered user list.
  6. Block and Unblock: Admin can block and un-block any user for a spesific time duration.
  7. Requist List: Admin can see user request list jo user ne send ki thi product k liye, and also see the user detail whice send request.
